## ГЛАВНЫЕ ВЫВОДЫ (зафиксировано)

1. **Сервер думает, телефон исполняет.** Сервер (раз в 2 ч): живость, гео, тип IP,
   стабильность → JSON-рейтинг. Телефон: скорость/пинг (спидтест), кнопки.
2. **AI без логина = страна + стабильность.** Держим на Германии (вывод 9).
3. **Проблема Gemini — репутация выходного IP под аккаунтом**, не тип IP. Решение —
   рабочий регион (Германия).
4. **MaxMind — зеркало P3TERX/GeoLite.mmdb без ключа.**
5. **РКН — два типа блокировок:** внутренние (Re-filter) и внешние (RU-IP режут →
   no-russia-hosts). Нужны оба.
6. **TikTok — БЕЗ MITM** (маршрутизация доменов).
7. **Авторизованная проверка AI отвергнута** (бан).
8. **AI-группа: Германия-якорь (по ИМЕНИ узла) + резерв**, только основные VPN,
   одна select-группа RH-AI + cron-селектор routehub-core.js.
9. **Двухслойная модель** (сервер/телефон).
10. **Конфиг — одна ссылка** (Worker, keyed). Токен GitHub на телефоне запрещён.
11. **Личные подписки (Этап D, завершён):** Worker по `?key=kN`, devices.json (free/
    bound + nonce), токен только на Worker.
12. **Оценка узлов — `ЭТАП_D_ФОРМУЛА.md`:** гейт сервера фильтрует, балл AIM ранжирует,
    иконка = тир (блок) + % от быстрейшего; кнопка — локальный штраф.
13. **СТРАНА УЗЛА — ИЗ ИМЕНИ (флаг 🇩🇪 ИЛИ слово), НЕ из GeoIP `rating.country`**
    (GeoIP выходного IP врёт: финский узел с country=DE и т.п.). `countryFromName` в
    core/health/netwatch/ai-bad.
14. **Переключение группы — `$config.setSelectPolicy` (работает); `getConfig(g,n)` —
    читающий.** async/await в Loon (cron/network-changed/generic) — работают.
    Cron-таймер сбросить нельзя — синхронизация скриптов через `rh_ai_checked` + lock.
