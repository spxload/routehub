## ГЛАВНЫЕ ВЫВОДЫ (зафиксировано, не пересматривать без причины)

1. **Сервер думает, телефон исполняет.** Тяжёлая проверка (AI,
   гео, тип IP, скорость) — на сервере раз в 2 ч → JSON-рейтинг.
   Loon читает рейтинг и переключает группы. Подтверждено
   исследованием — так делают все сильные проекты.

2. **AI-доступность без логина = только страна + тип IP +
   стабильность.** Реальный доступ под аккаунтом сервер НЕ видит
   (только телефон). Готового переключателя по ChatGPT/Claude нет
   нигде — пишем сами по паттерну «тест → setSelectPolicy».

3. **Проблема Gemini = тип IP.** Google жёстко банит датацентр/VPN
   даже в разрешённой стране. Определяется на сервере через ASN
   (X4BNet + is-vpn). Gemini хорош на residential, плох на
   datacenter.

4. **MaxMind — через зеркало P3TERX/GeoLite.mmdb БЕЗ ключа**
   (регистрация на maxmind.com Диане недоступна). Уже внедрено.

5. **РКН — два типа блокировок:** внутренние (РКН режет наружу →
   Re-filter) и внешние (сервисы сами режут RU-IP → no-russia-
   hosts). Это РАЗНЫЕ списки, нужны оба.

6. **TikTok — БЕЗ MITM.** Semporia/TikTok-Unlock отвергнут (MITM,
   устарел, риск бана). Вместо него — маршрутизация доменов TikTok
   на узел нужной страны без переписывания региона.

7. **Авторизованная проверка AI по узлам отвергнута окончательно**
   (impossible-travel-бан, небезопасно).
